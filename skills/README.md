# epicsagas/skills

Personal agent skill collection — problem discovery, cognitive self-analysis, and OSS distribution readiness.

## Skills

| Skill | Description | Triggers |
|-------|-------------|----------|
| **discover** | Problem discovery for individuals, teams, and startups. Reframes vague ideas into structured problem definitions (5 Whys, JTBD, Fishbone, Socratic, etc.) | Vague requests, solution-first thinking, "what should we build" |
| **cognitive-audit** | Evidence-based analysis of thinking patterns from conversation history — cognitive biases, decision-making style, actionable routines | "cognitive audit", "analyze my thinking", "my biases" |
| **oss-dist** | OSS project release readiness across the full lifecycle — community standards, README, launch strategy, distribution pipeline, i18n, security | "is this ready to open-source", "how do I distribute", "write a README" |

## Install

```bash
# Claude Code
claude plugin marketplace add epicsagas/plugins
claude plugin install epicsagas@epicsagas

# Codex CLI
codex plugin marketplace add epicsagas/plugins
codex plugin add epicsagas@epicsagas

# Antigravity
agy plugin import claude
agy plugin enable epicsagas
```
